Git Reference

Add a new repo to github:
git remote add origin https://github.com/ToddBowers/repoName.git

Add an existing project to GitHub:
1. Change the directory to your current local project.
2. $git init
3. $git add .
4. git commit -m "First commit"
5. git remote add origin https://github.com/ToddBowers/repoName.git
6. git push origin master


Clone from a remote repository:
1. $git clone https://github.com/toddbowers/<repoName>.git

Clone a directory from a remote repository:
1. $git clone https://github.com/toddbowers/<repoName>.git/<directory name>???

Setting up a repository:

Notes: Central repositories should always be created as "--bare" repositories.

Configure git:

$git config --global user.name <name>
$git config --global user.email <email>
$git config --global alias.<alias-name> <git-command>
$git config --system core.editory <editor>
